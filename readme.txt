Exiba uma Linda Vitrine de Produtos que você é Afiliado no Hotmart ou outro site qualquer. Gerencie tudo pelo Painel do Plugin. 


== Description ==

**Apresentação**

Com este plugin você pode criar uma vitrine de produtos que você é afiliado e exibi-la em seu site facilmente. 

Cadastre infinitas vitrines e exiba em quaisquer posts do seu blog que você desejar.

O plugin trabalha com shortcode, basta colocar o shortcode da vitrine na página ou post do seu blog para que a Vitrine seja exibida.

https://www.youtube.com/watch?v=s_b0VH_lptY


**Sobre**

O plugin Vitrine Hotmart foi escrito por Anderson Makiyama do [Plugin Wordpress](http://plugin-wp.net)


== Installation ==

Para instalar e usar o plugin é muito simples:

1- Envie a pasta do plugin para dentro da pasta de plugins, que geralmente é /wp-content/plugins/

2- Faça login no painel de gerenciamento do teu blog WP, vá na seção plugin e ative o Plugin Hot Vitrine

3- Agora, no lado esquerdo do painel do blog, uma nova opção chamada "Hot Vitrine" aparecerá, click sobre esta nova opção para ir para a página de opções.

4- Na página de opções você pode ver as vitrines, adicionar novas, adicionar produtos, etc.


Para exibir a Vitrine, em qualquer página ou post do blog basta colocar as shortags da Vitrine e pronto.
